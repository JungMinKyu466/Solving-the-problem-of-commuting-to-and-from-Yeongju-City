# 📹 영천의 미학: 영천시 1km 격자 기반 복합 취약 지역 도출 및 인프라 사각지대 분석

**영천의 미학**은 영천의 지역별 인프라 격차 및 사회적 고립 문제를 영천시 고령화 인구,생활용 지하수,소방/의료/버스정류장 위치 데이터를 기반으로 분석하고, 복합 취약 지역 파악 후 개선 우선 순위 구역을 제안하는 프로젝트입니다.

---

## 🎯 프로젝트 목표

저희 영천의 미학 팀은 다음과 같은 목표를 바탕으로 프로젝트를 진행하였습니다.

- 고령 인구 분포, 생활용 지하수 사용 분포, 의료/소방/버스정류장 위치 분포를 시각화
- 격자 별로 각 계수의 취약 지표 도출 후 지표 간 상관계수 파악
- 복합 취약 지수 도출(가중치 모델링) 후 복합 취약 지역 시각화
- 취약 유형화 후 대시보드를 통해 취약 지역 유형 별 정책 제안

---

## 📚 데이터 및 분석 방법론

저희는 총 **7종의 csv,shp파일**를 수집 및 전처리하여 분석에 활용하였습니다.

- 영천시 1K 격자 데이터
- 영천시 1K 인구 데이터
- 영천시 1K 주택 데이터
- 영천시 1K 생활용 지하수 데이터
- 영천시 소방시설 데이터
- 영천시 병원시설 데이터(병원/응급실/보건소)
- 영천시 버스정류장 데이터

이후 다음과 같은 기법을 적용하였습니다.

- 사용 라이브러리:Geopandas,Pandas,Numpy,Matplotlib,ckDTree,Osmx,Networkx
- 선형 회귀->우회 계수 산출 모델링

  문제점: Tmap API로 거리를 산출하지 못하는 데이터 다수 발생(영천시 지형 특성=산지,농로 많음 영향)

  해결 방안: 거리를 산출하지 못한 경우 유클리드거리 * 우회계수를 거리로 대체

  목표: 영천시 지형에 가장 적합한 우회 계수를 산출

  방법: Tmap API를 통해 구한 실제 도보거리 값과 유클리드 직선거리 값을 선형 회귀로 분석하여 영천시 맞춤형 우회 계수 산출. 이때 상하위 5%의 이상치를 제거함.

  의의: 다양한 우회 계수 산출법으로 교차 검증.
  기존처럼 격자 중심점 대신 영천시 '도로망'위 무작위 좌표 추출하여 주변 정류장과의 도보 소요시간 재호출하는 방식으로 우회 계수 산정->한계 발생(여전히 너무 많은 이상치 발생)->선형회귀로 모델링한 우회 계수의 타당성 확보

  결론:영천시 최적 우회계수=2.07

- 히트맵 분석->대중교통/소방시설/의료시설 접근성 취약 지수를 시각화

  스무딩 기법->격자 이상치 처리. 주변 격자들의 평균값으로 대체

- 엔트로피 가중치 분석기법+AHP (예정)
